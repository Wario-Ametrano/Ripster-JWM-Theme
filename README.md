### Dark and Minimal theme for JWM.

This is a dark and minimal theme for the JWM desktop environment. This theme was designed to be both functional and to have the best performance on JWM, in fact it does not use any particular external package.

![Dark](https://github.com/Wario-Ametrano/Ripster-JWM-Theme/blob/main/Ripster-JWM-Desktop.png "Ripster-Theme")

#### Requirements:

- **jwm**
- **Papirus** icon set.

## Installation:
- Install JWM Desktop environment:
```
sudo apt install jwm
```
- Clone the repository:
```
git clone https://github.com/Wario-Ametrano/Ripster-JWM-Theme.git
```
- Paste the Ripster theme .jwmrc file into your ***home*** page and replace it with the default file:
```
cp ~/Downloads/.jwmrc ~/
```

## Set theme and Reload JWM:
write the following command in your terminal: 
```
jwm -restart
```
